### `SYSDATE`

현재의 연, 월, 일, 시, 분, 초 반환

```sql
SELECT SYSDATE FROM DUAL; # 2025-11-10 10:48:28
```

### `EXTRACT(특정 단위 FROM 날짜 데이터)`

(MSSQL에서는 DATEPART)

```sql
SELECT EXTRACT (MONTH FROM SYSDATE) AS MONTH FROM DUAL; # MONTH 컬럼 밑에 11
```

### `ADD_MONTHS(날짜 데이터, 특정 개월 수)`

(MSSQL에서는 DATEADD)

```sql
SELECT ADD_MONTHS(TO_DATE('2025-12-31', 'YYYY-MM-DD', -1) AS PREV_MONTH FROM DUAL;
# PREV_MONTH 컬럼 밑에 2025-11-30 00:00:00
```
