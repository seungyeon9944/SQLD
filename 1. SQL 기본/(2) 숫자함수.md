### `ABS`

절댓값 반환

```sql
SELECT ABS(-1) FROM DUAL; # 1
```

### `SIGN`

수의 부호 반환. 

양수 → 1 / 음수 → -1 / 0 → 0

```sql
SELECT SIGN(-7) FROM DUAL; # -1
```

### `ROUND(수 [,자릿수])`

**자릿수까지 반올림**

음수면 **-1 → 십의자리까지** 반올림 / **-2 → 백의자리까지** 반올림

*✏️ ROUND(2024-02-20 14:00:00, ‘MONTH’)하면 2024-03-01 00:00:00 됨*

```sql
SELECT ROUND(163.76, 1) FROM DUAL; # 163.8
SELECT ROUND(163.76, -2) FROM DUAL; # 200
```

### `TRUNC(수 [,자릿수])`

**자릿수까지 버림**

음수면 -1 → 십의자리까지 버림 (n0 이런식으로) / -2 → 백의자리까지 버림

```sql
SELECT TRUNC(54.29, 1) FROM DUAL; # 54.2
SELECT TRUNC(54.29, -1) FROM DUAL; # 50
```

### `CEIL(수)`

소수점 이하의 수 **올림** (무조건 정수로) → 수보다 크거나 같은 최소 정수 !

✏️ *음수는 소수점 버려서 수 자체는 올림 !*

(MSSQL에서는 CEILING)

```sql
SELECT CEIL(106.94) FROM DUAL; # 107
SELECT CEIL(-94.106) FROM DUAL; # -94
```

### `FLOOR(수)`

소수점 이하의 수 **버림 →** 수보다 작거나 같은 최대 정수 !

```sql
SELECT FLOOR(106.94) FROM DUAL; # 106
SELECT FLOOR(-94.106) FROM DUAL; # -95
```

### `MOD(수1, 수2)`

수1을 수2로 나눈 나머지

```sql
SELECT MOD(-15,-4) FROM DUAL; # -3
SELECT MOD(15, 0) FROM DUAL; # 15
```

### ⭐VARCHAR와 CHAR 비교

| **VARCHAR(5)** | **CHAR(5)** |
| --- | --- |
| 가변형`'SQL'` | 고정형 - **뒤에 공백 삽입** `'SQL  '` |
|  | **뒷자리 공백 제거** 후 비교 `'ab' = 'ab  '` |

✏️ *CHAR, VARCHAR인 경우 데이터가 있어도 서로 변경가능. 굳이 빈 컬럼일 필요x*
