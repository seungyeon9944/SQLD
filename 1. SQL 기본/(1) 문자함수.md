### `CHR`

ASCII 코드 (MSSQL에서는 `CHAR` )

```sql
SELECT CHR(65) FROM DUAL; # A
SELECT CHR(97) FROM DUAL; # a
```

### `LOWER` / `UPPER`

문자를 소문자 / 대문자로 변환

### `LTRIM(문자열 [,특정문자])`  / `RTRIM(문자열 [,특정문자])`

(MSSQL에서는 공백제거만 가능)

```sql
SELECT LTRIM('    NCT') FROM DUAL; # NCT
SELECT LTRIM ('NCT MARK', 'NCT') FROM DUAL; # MARK

SELECT RTRIM('MARK     ') FROM DUAL; # MARK
SELECT RTRIM('NCT MARK', 'MARK') FROM DUAL; # NCT

# COL1이 'SQL'이고 COL2가 'DATABASE'일 때
SELECT LENGTH(RTRIM(COL1, 'LE')) + LENGTH(LTRIM(COL2, 'DE')) AS RESULT 
# 'SQ'와 'ATABASE'여서 답은 9
# ✏️'D' 다음에 'E'가 아닌 'A를 만나서 작업멈추고 남은문자 다 반환
```

### `TRIM([위치] [특정문자] [FROM] 문자열)`

같으면 제거, 같지않으면 멈춤. **특정문자 한 글자만 가능**

(MSSQL에서는 공백제거만 가능)

```sql
SELECT TRIM('   NCT ') FROM DUAL; # NCT
SELECT TRIM(LEADING 'M' FROM 'MARK') FROM DUAL; # ARK
SELECT TRIM(TRAILING 'K' FROM 'MARK') FROM DUAL; # MAR
```

### `SUBSTR(문자열, 시작점 [,길이])`

원하는 부분만 잘라서 반환

[,길이] 없으면 시작점부터 쭉 다 반환 !

(MSSQL에서는 SUBSTRING(문자열))

```sql
SELECT SUBSTR('NCTMARK', 4, 4) FROM DUAL; # MARK
```

### `LPAD(문자열, 길이, 문자)`

설정한 길이가 될 때까지 왼쪽을 문자로 채움

```sql
SELECT LPAD('MARKLEE',10,'A') FROM DUAL; # AAAMARKLEE
```
